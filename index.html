When triggered, the app automatically sends an SMS to the preset emergency contacts, containing crucial information such as the user's live location, precise address, and a link to visual/audio evidence(automatically captured by pressing the button) stored securely on a cloud platform like Google Drive.


2.Laser Security System with IoT alerts

First, the security system will be connected to the home Wi-Fi. If the laser falling on the LDR is blocked by any person, the resistance of the LDR will increase, sending '1' as input to the ESP8266. This will trigger the LED and buzzer to turn on, and a message "Warning! Someone is entering the house!" will be sent to the user's Telegram.
